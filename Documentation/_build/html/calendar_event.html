calendar_event module¶

class calendar_event.calendar_event¶

Bases: object

The calendar_event class is used to add or delete events for return of books from google calendar, based on whether a book has been borrowed or returned.

SCOPES = 'https://www.googleapis.com/auth/calendar'¶
creds = None¶
delete(book_id)¶

This method deletes a google calendar event for the return of a book when the book is returned, using the book id to match the event.

Arguments:

book_id {int} – The id of the book returned

insert(book_title, book_id)¶

This method inserts a new google calendar event for the return of a book whenever a book is borrowed. The event is added one week from the current date.

Arguments:

book_title {string} – The title of the book to be specified in return book google calendar event. book_id {int} – The id of the book to be specified in return book google calendar event.
