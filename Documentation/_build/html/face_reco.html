class face_reco.face_reco¶

Bases: object

The face_reco class is used to identify a user’s face for login purposes.

detect_face()¶

This function is used to identify a user’s face for login purposes. A video stream is initiated after which the user’s face is matched with the encoded pickle file. If a match is found, the username of the user is returned. If no match is found, Unknown will be returned.

Returns:

string – Either username of the user whose face has been identified or Unknown.
